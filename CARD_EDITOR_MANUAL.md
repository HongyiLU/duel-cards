## 🚀 快速开始

### 5步配置卡牌效果

```
步骤 1️⃣ → 选择卡牌
步骤 2️⃣ → 选择触发时机
步骤 3️⃣ → 添加效果
步骤 4️⃣ → 调整数值和目标
步骤 5️⃣ → 保存配置
```

### 30秒上手视频脚本

1. 在左侧卡片列表点击「圣光骑士」
2. 在中间区域点击「战吼」触发时机
3. 点击「⚡ 效果模板」选择「解场」模板
4. 将伤害值调整为 3
5. 点击右上角「💾 保存配置」

## 📝 基础操作

### 选择卡牌

1. **通过搜索选择**
   - 点击搜索框
   - 输入卡牌名称（如「骑士」）
   - 点击匹配的卡牌

2. **通过分组选择**
   - 点击「生物卡」或「法术卡」展开分组
   - 在网格中点击想要的卡牌
   - 选中后显示金色边框 ✓

### 配置触发时机

1. 触发时机按钮位于「卡牌预览」下方
2. 可选时机：
   - 📢 **战吼** - 打出卡牌时
   - ⚔️ **攻击** - 攻击时
   - 💀 **亡语** - 死亡时
   - 🌅 **回合开始** - 你的回合开始时
   - 🌙 **回合结束** - 你的回合结束时
3. 点击选中，金色高亮表示当前选择

### 添加效果

#### 方法一：使用模板（推荐）
1. 点击「⚡ 效果模板」展开模板列表
2. 选择想要的模板（如「💚 治疗」）
3. 模板自动添加到效果列表

#### 方法二：手动添加
1. 点击「➕ 添加新效果」
2. 选择效果类型
3. 调整数值
4. 选择目标

### 编辑效果

1. **展开/收起效果**
   - 点击效果卡片标题行
   - ▶ 收起 / ▼ 展开

2. **修改效果参数**
   - **类型**: 下拉选择不同效果
   - **数值**: 使用 +/- 按钮或直接输入
   - **目标**: 下拉选择目标类型

3. **调整效果顺序**
   - 点击 ↑ 按钮上移
   - 点击 ↓ 按钮下移
   - 效果按从上到下顺序执行

4. **删除效果**
   - 点击 ✕ 按钮删除该效果

### 保存配置

1. 配置完成后，点击「💾 保存配置」
2. 弹出「✅ 效果配置已保存！」提示
3. 配置数据传递给 `onSave` 回调

## 📢 触发时机说明

### 战吼 (ON_PLAY)
- **图标**: 📢
- **触发时间**: 卡牌被打出时
- **典型用法**: 绝大多数卡牌效果
- **示例**: 「战吼：对敌方随从造成3点伤害」

### 攻击 (ON_ATTACK)
- **图标**: ⚔️
- **触发时间**: 卡牌攻击时
- **典型用法**: 冲锋、攻击时触发的效果
- **示例**: 「攻击：对攻击目标造成额外2点伤害」

### 亡语 (ON_DEATH)
- **图标**: 💀
- **触发时间**: 卡牌死亡时
- **典型用法**: 死亡时触发的遗留效果
- **示例**: 「亡语：随机对一个敌方随从造成3点伤害」

### 回合开始 (ON_TURN_START)
- **图标**: 🌅
- **触发时间**: 你的回合开始时
- **典型用法**: 持续性/周期性效果
- **示例**: 「回合开始：抽一张牌」

### 回合结束 (ON_TURN_END)
- **图标**: 🌙
- **触发时间**: 你的回合结束时
- **典型用法**: 回合结束触发的效果
- **示例**: 「回合结束：获得1点护甲」

## ⚡ 效果模板

### 什么是效果模板？

效果模板是预设的效果组合，点击即可快速添加，无需手动配置。

### 可用模板

| 模板 | 图标 | 配置 | 说明 |
|------|------|------|------|
| 打脸 | 🎯 | 伤害 3 → 敌方英雄 | 直伤打脸 |
| 解场 | ⚔️ | 伤害 3 → 敌方随从 | 单体解场 |
| AOE | 💥 | 全体伤害 2 → 所有敌方随从 | 群体解场 |
| 治疗 | 💚 | 治疗 3 → 友方英雄 | 回复生命 |

## ❓ 常见问题

### Q: 为什么会自动切换目标？

A: 当你切换「效果类型」时，系统会根据该效果类型自动设置「默认目标」。例如：
- 选择「造成伤害」→ 目标自动设为「敌方随从」
- 选择「治疗」→ 目标自动设为「友方英雄」

### Q: 可以添加多个效果吗？

A: 可以！点击「➕ 添加新效果」或使用模板添加多个效果，效果按从上到下顺序执行。

### Q: 如何调整效果执行顺序？

A: 使用效果卡片右侧的 ↑ ↓ 按钮调整顺序。上方的效果先执行。

### Q: 效果描述是如何生成的？

A: 系统根据你选择的触发时机、效果类型、数值和目标自动生成描述文本，显示在右侧预览区。

### Q: 搜索不到想要的卡牌？

A: 检查：
1. 搜索词拼写是否正确
2. 卡牌是否在数据库中（需要先在 cardDatabase 添加）
3. 尝试使用部分名称搜索

### Q: 模板的数值可以修改吗？

A: 可以！模板只是快速添加，添加后你可以修改任何参数（类型/数值/目标）。

## 🎮 技巧

### 1. 快速复制效果
如果需要相似的多个效果，先添加一个，修改后使用「添加新效果」创建副本。

### 2. 利用效果描述预览
配置时注意看右侧「📜 效果描述」，确保生成的效果描述符合你的设计意图。

### 3. 触发时机选择
- 绝大多数卡牌用「战吼」(ON_PLAY)
- 亡语卡用「亡语」(ON_DEATH)
- 辅助型卡牌用「回合开始/结束」

### 4. 组合效果
可以添加多个效果，例如：
```
效果1: 战吼 - 造成3点伤害 → 敌方随从
效果2: 战吼 - 治疗2点 → 友方英雄
```
效果描述会合并为：「战吼：对敌方随从造成3点伤害，为你恢复2点生命」
