# 卡牌效果编辑器优化方案 v1.1

**日期**: 2026-03-23
**版本**: v1.1
**状态**: ✅ 已完成

---

## 1. 需求分析

### 1.1 当前问题
用户反馈「效果编辑器」交互方式不够直观，主要问题：
1. ❌ 左侧卡牌列表只有文字，缺少视觉预览
2. ❌ 效果配置界面简陋，不易理解
3. ❌ 缺少触发时机选择（ON_PLAY/ON_ATTACK/ON_DEATH等）
4. ❌ 没有效果组合的视觉化展示
5. ❌ 操作手册缺失

### 1.2 优化目标
- ✅ 让卡牌效果配置像搭积木一样直观
- ✅ 一眼看清效果预览
- ✅ 完整的操作手册

### 2.2 新增功能

#### 2.2.1 效果快捷模板
```
常用效果组合模板:
├── 💥 打脸输出
│   └── 造成 X 点伤害 → 敌方英雄
├── ⚔️ 解场
│   └── 造成 X 点伤害 → 敌方随从
├── 💚 自奶
│   └── 治疗 X 点 → 友方英雄
├── 🛡️ 护盾
│   └── 获得 X 护甲 → 友方英雄
└── 🎴 过牌
    └── 抽 X 张牌 → 自己
```

点击模板自动填充效果配置

#### 2.2.2 效果条件配置（高级）
```
触发条件 (可选):
├── 无条件
├── 当生命值 ≤ X 时
├── 当手牌数 ≤ X 时
├── 当回合数 ≥ X 时
└── 当场上随从数 = X 时
```

## 4. 操作手册内容

### 4.1 基础操作
1. 选择要编辑的卡牌
2. 选择触发时机
3. 添加/配置效果
4. 预览效果描述
5. 保存配置

### 4.2 效果类型说明
| 图标 | 类型 | 说明 |
|------|------|------|
| ⚔️ | 造成伤害 | 对目标造成 X 点伤害 |
| 💚 | 治疗 | 恢复目标 X 点生命 |
| 🛡️ | 获得护甲 | 获得 X 点护甲 |
| 🎴 | 抽牌 | 抽 X 张牌 |
| ❄️ | 冻结 | 目标跳过 X 回合 |
| 💥 | 全体伤害 | 对所有敌方随从造成 X 伤害 |
| 🎯 | 英雄伤害 | 对敌方英雄造成 X 伤害 |
| ⚡ | 获得法力 | 本回合获得 X 点额外法力 |
| 💀 | 消灭 | 消灭一个目标（无视生命） |
| 📋 | 复制卡牌 | 复制一张敌方手牌 |
| 🔇 | 沉默 | 移除目标所有效果 |

### 4.3 目标类型说明
| 类型 | 说明 |
|------|------|
| 自身 | 效果作用于使用此牌的卡牌 |
| 敌方随从 | 选择一个敌方随从 |
| 所有敌方随从 | 对所有敌方随从生效 |
| 所有友方随从 | 对所有友方随从生效 |
| 友方英雄 | 你自己 |
| 敌方英雄 | 对方玩家 |
